Installation

아래 링크를 통해 daARA Github로 이동하여 daARA PROJECT 프로그램을 설치합니다.

 Usage

1. 출석 체크를 적용할 강의 혹은 이미지를 준비합니다.

2. 다운받은 daARA PROJECT File에서 다아라 출석 체크 프로그램을 실행합니다.

3. 초기 이용시 회원가입을 한 후 로그인을 한 후 프로그램 사용이 가능합니다.

4. 강의관리 → 강의 추가하기를 통해 강의를 추가하고 해당 강의를 수강하는 학생들을 추가합니다.

5. 강의 시작하기 탭에서 강의를 시작합니다.
